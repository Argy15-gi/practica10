Tipos de Amenazas

Conoce los principales vectores de ataque en el ciberespacio

Malware

Software malicioso diseñado para dañar, infiltrarse o dañar sistemas informáticos.

  • Virus
  • Troyanos
  • Ransomware
  • Spyware
Nivel de Riesgo:

Phishing

Técnicas de engaño para obtener información confidencial haciéndose pasar por entidades de confianza.

  • Spear Phishing
  • Whaling
  • Smishing
  • Vishing
Nivel de Riesgo:

DDoS

Ataques de denegación de servicio distribuidos que saturan sistemas para hacerlos inaccesibles.

  • Volumétricos
  • Protocolo
  • Aplicación
  • Amplificación
Nivel de Riesgo:

Inyección SQL

Aprovechamiento de vulnerabilidades en bases de datos mediante código malicioso SQL.

  • SQLi Clásico
  • Blind SQLi
  • Union Based
  • Error Based
Nivel de Riesgo:

XSS

Cross-Site Scripting permite inyectar scripts maliciosos en páginas web vistas por otros usuarios.

  • Reflejado
  • Almacenado
  • DOM Based
  • Self-XSS
Nivel de Riesgo:

Man in the Middle

Ataques donde el atacante intercepta y posiblemente altera la comunicación entre dos partes.

  • ARP Spoofing
  • DNS Spoofing
  • SSL Stripping
  • Session Hijacking
Nivel de Riesgo:
